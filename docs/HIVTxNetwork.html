simplify_multisequence_cluster()

When MSPP are present, this function will reduce the network encoded by .Nodes and .Edges in filtered_json, and reduce all sequences that represent the same entity into one node. Such nodes inherit the union of their links (so at least of the sequences being collapsed link to X, the "joint" node will link to X). The joint nodes will also receive aggregated attributes; if the nodes being merged have different attributes values for a given key, the merged node will have a ';' separated list of attributes for the same key.
